Submitting a new tutorials

You can help Scratch Tutorials by submitting a new tutorial here, reading the instructions given on the page and pasting the content here.

You can help by helping with the design, for example by suggesting some ideas for a fresh and modern new page layout, or something not very important to clean the mobile friendly interface, and you can still imagine something you’d like to add. With some luck, it will be added.
